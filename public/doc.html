Configuration

Flash is very simple to use and right now allows using into two modes,
configurable delay and configurable delay and URL


Configurable Delay

If only delay parameter is passed on, Flash responds with a sample JSON response after the delay time period has elapsed. You can pass a time duration (in milliseconds) or pass 'random' for a random delay.
Allowed Delay Params Sample URL Value
time in millisecs /delay/4000 any integer value in millisecs
random /delay/random 'random' (return random time between 0 and 1000ms)

Configurable Delay and URL

You can get a delayed response from a specific API endpoint by passing an URL parameter to Flash. Along with the URL, you can still pass on a delay parameter with either of the configurations metioned above.
Allowed URL Params Sample URL Value
domain name /delay/4000/url/google.com any valid domain name, by default http is appended while redirecting
URL /delay/4000/url/https://www.google.com any valid domain name prepended by http:// or https://)

About Flash and Shout-outs

Flash is a tiny app whose goal is to make you write lesser lines of code while mocking a slow API response. It is built on NodeJS with Electron Framework.
